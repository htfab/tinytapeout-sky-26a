## How it works

Utilises a combination of XOR and NXOR gates to make the first letter of my name 'Z' appear with the correct code.

## How to test

Toggle switches on, off, on, on, on, off, on, off for IN0, IN1, IN2, IN3, IN4, IN5, IN6, IN7 respectivley to see the letter 'Z' appear.
